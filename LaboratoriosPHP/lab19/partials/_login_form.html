¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications?

[1] Puedes leer y enviar datos desde el servidor y actualizar una página sin necesidad de recargarla

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

[2] Los principales problemas de seguridad se ven del lado del usuario ya que el código JavaScript siempre es visible para los clientes, que pueden inferir debilidades del servidor.
Las validaciones se deberán hacer en el lado del servidor para no se hagan consultas maliciosas del lado del cliente.

¿Qué es JSON?

[3] JavaScript object notation, es un formato para el intercambios de datos, básicamente JSON describe los datos con una sintaxis dedicada que se usa para identificar y gestionar los datos.
